# Ansible-Playbook

## Description

Building Nginx + PHP 7 + mariadb + Let's encrypt (certbot) + Wordpress environment for clients

## Requirements.

### Host
- Ansible 2.2

### Client
- CentOS 7.3

## Usage

First, you need edit `playbook.yml`,`group_vars/all`

```bash
$ ansible-playbook playbook.yml
```
